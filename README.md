# SGTA - Sistema de Gerenciamento de Tarefas Acadêmicas

API REST desenvolvida em Python e Django.

Funcionalidades:
- CRUD de usuários
- CRUD de tarefas
- Controle de status
- Relacionamento usuário → tarefas

1 -
    - models.py -> Tabelas(dados) que vamos usar 
    - views.py -> Pega esses dados
    - urls.py -> São as rotas
    -admin.py -> adiministrador
    -apps.py -> Configurações do app
